#Arithmetic Operator 

- Arithmetic operator are used to perform mathmatical operation in Java.

##Arithmetic operator 
- Addtion '+'
- Subtract '-'
- Multiply '*'
- Division '/'
- Modules '%' (reminder)

##For example: 

## Example

```java
int a = 10;
int b = 20;

// Addition
System.out.println(a + b);
```

Output:

```java
30
```

---

```java
// Subtraction
System.out.println(b - a);
```

Output:

```java
10
```

---

```java
// Multiplication
System.out.println(a * b);
```

Output:

```java
200
```

---

```java
// Division
System.out.println(b / a);
```

Output:

```java
2
```

---

```java
// Modulus
int x = 13;
int y = 2;

System.out.println(x % y);
```

Output:

```java
1
```
